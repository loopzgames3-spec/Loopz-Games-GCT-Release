# 🎮 Loopz Games GCT — Game Creation Tool

Loopz Games GCT is a free 2D game creation tool for Windows. Build your own games with no coding experience required!

## ⬇️ Download

👉 **[Click here to download the latest version](https://github.com/loopzgames3-spec/Loopz-Games-GCT-Release/releases/tag/v1.0.0)**

No installation needed — just download and run the `.exe`!

---

## ✨ Features

- 🖼️ Sprite Import & Frame-based Animation System
- 🎬 Cutscene & Dialogue System
- 🗺️ Drag-and-drop Tilemap Editor
- 🎲 Board Game Template
- 🎮 8 Game Templates (RPG, Platformer, Roguelike, Puzzle, Shooter…)
- 💻 VS Code–style Script Editor with syntax highlighting
- 🎨 10 UI Themes
- 🌍 Multi-language Support
- 🔄 Built-in Auto-updater

## 🚀 Getting Started

1. Download `LoopzGamesGTC.exe` from the [Releases page](https://github.com/loopzgames3-spec/Loopz-Games-GCT-Release/releases/tag/v1.0.0)
2. Double-click to run — no installation needed
3. Check for updates anytime via **Help → Check for Updates**
